# ⛔️ `astro-ai-robots-txt`

Block AI scrapers in the `robots.txt` for your Astro site

## Quick start

```sh
npx astro add astro-ai-robots-txt
```

See the [docs site](https://delucis.github.io/astro-ai-robots-txt) for more details.
